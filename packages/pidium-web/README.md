# @pi-nema/absolute-web

Keyless web search and content fetching for `pi`.

## Tools

- `web_search`: searches DuckDuckGo HTML first and falls back to Exa MCP.
- `fetch_content`: fetches URLs and extracts markdown, text, or HTML.
- `get_web_content`: retrieves full stored content from previous calls.

Install from the repository root:

```bash
pi install ./packages/absolute-web
```

No API keys, browser login, or local services are required for the default path.
